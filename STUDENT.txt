# ============================================
# DESKRIPSI PROJECT
# ============================================

StoryGlow adalah Progressive Web Application (PWA) untuk platform berbagi cerita yang memungkinkan pengguna berbagi pengalaman dan cerita menarik dari seluruh dunia.
Aplikasi ini dilengkapi dengan fitur peta interaktif, push notification, offline support, dan background sync untuk pengalaman pengguna yang optimal.

# ============================================
# KNOWN LIMITATIONS
# ============================================

- Push notification tidak work di Safari (iOS/macOS)
- Background Sync API tidak support di Firefox (fallback: manual sync)
- Camera capture mungkin tidak work di browser tertentu (fallback: file upload)
- Geolocation permission required untuk "Lokasi saya saat ini"
- IndexedDB limit per domain: ~50MB (lebih dari cukup untuk use case ini)
- Service Worker cache size: unlimited tapi perlu periodic cleanup
